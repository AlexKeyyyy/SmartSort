# SmartSort: Шпаргалка Для Защиты

Короткая версия, которую можно открыть за 5 минут до выступления.

## 1) 20-секундное объяснение проекта

"SmartSort — desktop-приложение для автоматической сортировки файлов. Оно следит за папкой, классифицирует новые файлы по 8 категориям с помощью ML-модели и переносит их в нужные директории. Есть лог в SQLite и Undo."

## 2) Минимальные термины (простое + академичное)

- `TF-IDF`
  - просто: важность слова в документе с учетом редкости в корпусе
  - академично: взвешивание токенов по частоте в документе и обратной частоте в коллекции
- `n-gram`
  - просто: куски текста длины n
  - академично: локальные последовательности символов/слов для устойчивых признаков
- `MLP`
  - просто: нейросеть из полносвязных слоев
  - академично: feed-forward multilayer perceptron для многоклассовой классификации
- `backpropagation`
  - просто: способ научить сеть, уменьшая ошибку
  - академично: вычисление градиентов функции потерь и обновление весов
- `early stopping`
  - просто: остановка обучения, когда модель перестает улучшаться
  - академично: регуляризация через прекращение оптимизации по валидационному критерию
- `precision/recall/F1`
  - просто: точность, полнота и их баланс
  - академично: метрики качества классификатора по классам
- `confusion matrix`
  - просто: таблица, кто с кем путается
  - академично: матрица совместного распределения true/predicted классов
- `confidence threshold`
  - просто: минимальная уверенность для автоматического решения
  - академично: порог вероятности для постобработки предсказаний

## 4) Что показать руками

1. `python main.py`
2. Вкладка Watcher: `Start`
3. Добавить тестовые файлы из `data/demo_pack/input`
4. Показать `confidence` и категории
5. `Apply all`
6. Вкладка Log + кнопка `Undo`
7. Открыть `reports/model_eval/latest/summary.json` и `confusion_matrix.csv`
8. Открыть `reports/smoke/latest/smoke_report.json`

## 5) Частые вопросы преподавателя

- "Почему это нейросети?"
  - Потому что классификатор — `MLPClassifier` (многослойная нейросеть), обученный на текстовых признаках TF-IDF.
- "Почему не только правила?"
  - Правила работают только в узких случаях. Нейросеть обобщает паттерны из данных.
- "Почему высокая accuracy не гарантирует идеальность?"
  - Нужен анализ ошибок: можно иметь высокую accuracy и слабые места в отдельных классах.
- "Как вы подтверждаете стабильность системы?"
  - Через reproducible артефакты оценки + `smoke_checks`.
